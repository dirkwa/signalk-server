Help

How to use this plugin
  1. Upload an MBTiles file using the form above
  2. The chart will be available via Signal K Resources API
  3. Use Freeboard-SK or other chart clients to view the charts
  4. Charts are served at /plugins/charts-provider-go/tiles/{chartId}/{z}/{x}/{y}
Where to get MBTiles charts
  • OpenSeaMap - Free nautical charts
  • ChartWork - Chart conversion tools
  • GDAL - Convert various formats to MBTiles
